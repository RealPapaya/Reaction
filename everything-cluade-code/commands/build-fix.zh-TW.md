# Build and Fix

增量修復 TypeScript 與建置錯誤：

1. 執行建置：npm run build 或 pnpm build

2. 解析錯誤輸出：
   - 依檔案分組
   - 依嚴重性排序

3. 對於每個錯誤：
   - 顯示錯誤上下文 (前後 5 行)
   - 解釋問題
   - 提出修復方案
   - 套用修復
   - 重新執行建置
   - 驗證錯誤已解決

4. 停止條件：
   - 修復引入了新錯誤
   - 相同錯誤嘗試 3 次後仍存在
   - 使用者請求暫停

5. 顯示摘要：
   - 已修復的錯誤
   - 剩餘錯誤
   - 引入的新錯誤

為了安全起見，一次修復一個錯誤！
